Window Panels

The flo-window-panes component is a response grid window/tile manager.

Features

  • High Performance with minimal CPU and memory usage
  • Responsive for an arbitrary number of elements with a high degree of layout symmetry.
  • Primarily designed for a multi-camera grid layout, but arbitarty DOM elements should also work.
  • CSS based solution with minimal Javascript (for layout) to enable a server rendered and jitter free user experience
  • Drag and drop to swap element positions
  • Extensive API for handle to customize for broad use-cases
